## AI Tools

- [Claude](https://claude.ai) - Handles writing, coding, research, and complex reasoning well. Noticeably better than most alternatives for anything that requires nuanced thinking or longer context.
- [Cursor](https://cursor.sh) - A code editor with AI baked in at a deep level, not just autocomplete. Solo founders especially tend to get a lot of mileage out of it.
- [Perplexity](https://perplexity.ai) - AI search that cites its sources so you can actually verify what it tells you. Good for competitive research and quick fact-finding.
- [Whisper](https://github.com/openai/whisper) - Open-source speech recognition from OpenAI. Works well for transcribing user interviews and internal calls without sending everything to a third-party service.
- [Descript](https://descript.com) - You edit video and audio by editing the transcript like a document. Once you try it the old way feels very slow.
- [ElevenLabs](https://elevenlabs.io) - AI voice generation that actually sounds natural. Useful for demos and explainer videos before you're ready to invest in production.
- [Bolt](https://bolt.new) - Generates full-stack web apps from a prompt. Worth trying when you need to validate an idea quickly without writing boilerplate.
- [v0](https://v0.dev) - Generates React UI components from a text description or screenshot. A good way to get a solid starting point for a UI without staring at a blank file.

## Design

- [Figma](https://figma.com) - The standard collaborative design tool. The free plan covers most of what an early team needs.
- [Framer](https://framer.com) - Build a real, publishable website visually. A good landing page in Framer takes hours instead of days.
- [Mobbin](https://mobbin.com) - A library of real UI patterns from well-known apps. Useful when you're designing a flow and want to see how others have solved the same problem.
- [Unsplash](https://unsplash.com) - Free high-quality photos with no attribution required.
- [Iconify](https://iconify.design) - One place to access over 200,000 icons from open-source icon sets. Saves a lot of tab-switching.
- [Shots.so](https://shots.so) - Creates clean, beautiful mockups of your product for landing pages and social posts. Underused and very quick to set up.
- [Fontshare](https://www.fontshare.com) - Free, high-quality fonts from Indian Type Foundry. Better selection than Google Fonts for anything that needs a distinctive feel.

## Marketing and Growth

- [Loops](https://loops.so) - Email platform built for SaaS products. The workflow builder is more intuitive than most alternatives and it integrates cleanly with modern stacks.
- [Beehiiv](https://beehiiv.com) - Newsletter platform with solid built-in growth tools. Worth considering if content is a real part of your distribution strategy.
- [Plausible](https://plausible.io) - Privacy-friendly analytics that you can self-host. Much lighter than GA4 and the dashboard shows you what you actually need.
- [PostHog](https://posthog.com) - Product analytics, session recordings, feature flags, and A/B testing in one open-source tool. Replaces several separate products.
- [Cal.com](https://cal.com) - Open-source scheduling tool. A good Calendly replacement if you want to own your data and avoid paying per seat.
- [Tally](https://tally.so) - Free form builder that looks better than Typeform and has no response limits. Overlooked by a lot of teams.
- [Loom](https://loom.com) - Async video messaging. Replaces a lot of meetings and works well for onboarding new users or explaining features without writing a wall of text.

## Development and Infrastructure

- [Vercel](https://vercel.com) - Frontend deployment that mostly just works. The developer experience is hard to beat.
- [Supabase](https://supabase.com) - Open-source Firebase alternative with Postgres underneath. Includes auth, storage, and real-time out of the box.
- [Railway](https://railway.app) - Deploys backends without requiring a DevOps background. Good fit for early-stage teams that want to move fast without managing infrastructure.
- [Trigger.dev](https://trigger.dev) - Open-source background jobs and scheduled tasks with a proper developer experience. Much easier to reason about than setting up queues manually.
- [Sentry](https://sentry.io) - Error tracking. You want this running before launch so you hear about problems before your users do.
- [Resend](https://resend.com) - Email API with a developer-first design. Significantly more pleasant to work with than the older alternatives.
- [Clerk](https://clerk.com) - Authentication that takes an afternoon to set up properly. Handles all the edge cases so you don't have to.
- [Stripe](https://stripe.com) - Payments. Still the default choice and the docs are genuinely good.
- [SigNoz](https://signoz.io) - Open-source observability platform that covers logs, traces, and metrics in one place. A solid alternative to Datadog for teams watching their cloud spend.
- [Warp](https://warp.dev) - A terminal built for developers that adds AI assistance, shared sessions, and much better command history. Worth switching to if you spend a lot of time in the terminal.
- [Ngrok](https://ngrok.com) - Exposes your local server to the internet in one command. Essential for testing webhooks during development.
